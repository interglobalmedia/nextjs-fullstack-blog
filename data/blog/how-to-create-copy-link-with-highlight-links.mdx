The other day I found out how to link to specific text within a post on the web instead of simply linking to the entire post.

This approach is called `copy link with highlight` in `Safari`, and that is exactly what happens when a person clicks on that `special link`.

## How to create a copy link with highlight link in Safari

In order to create a `copy link with highlight` link in `Safari`, do the following:

- Select the text you would like to link to, and then `right-click` on the `selected text`. When you `right-click` on the `selected text`, the `copy link with highlight` option `appears` in the resulting `dropdown menu`:

![copy link with highlight example 1](Screenshot-2026-05-09-at-8.46.56-PM.jpg)

- `Copy` and `paste` the link wherever you want to `share` it. For example, you could share it in an `email` or `text` to a colleague or friend. Or you could `share` it on `social media`.

When you `click` on the `copy link with highlight` link, you are taken to something like the following:

![Copy link with highlight link result](Screenshot-2026-05-09-at-10.47.21-PM.jpg)

## Copy link to highlight in Chrome

In `Chrome`, the `feature` is called `copy link to highlight`.

You can create a `copy link to highlight` link in `Chrome` following the same steps executed in `Safari`.

## Copy link to highlight in Firefox

There is also a `copy link to highlight` feature in `Firefox`, which follows the same steps as in `Safari` and `Chrome`, but it `only works` in `Firefox Nightly Build`.

## Why copy link with highlight

`Sharing` a `copy link with highlight` link can be quite `beneficial`:

**Efficiency**: It `takes` an individual `clicking` on the `link` to a `specific spot` on a `web page` you want to "highlight" (pun intended). Perhaps there wasn't anything else on the page you are interested in sharing! This way, the other person does `not` have to `search` through the `page` to `find` the `relevant information`. This feature `speeds up` AND `enhances communication` and `collaboration` between `people`.
