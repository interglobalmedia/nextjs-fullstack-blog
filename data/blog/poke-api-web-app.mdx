_The Pokemon API App_

The `Pokemon API` app utilizes the `Poke API` based on `Pokemon characters` and
uses both the `fetch API` and `AJAX`, JavaScript `promise chaining`, the
`map method`, `creates` most `elements` rendered to the `html pages`, `checks`
for `non-existent` images and `removes` them or or `sets` their
`display properties` to `none`, uses `pointer events` to replace `touch` and
`mouse events`, `custom` build `script`, and `Modern JavaScript` in general.
