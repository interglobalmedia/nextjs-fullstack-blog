***Creating React boilerplate code in VS Code***

I **_just learned_** that **_not only_** can you **_create_** `boilerplate HTML`
in `VS Code`, but you can **_also create_** `boilerplate React code`
**_inside_** of a `React component` with the **_help_** of a `VS Code extension`
called
[ES7+ React/Redux/React-Native/JS snippets](https://marketplace.visualstudio.com/items?itemName=dsznajder.es7-react-js-snippets)
by `dsznajder`.

Once I **_installed_** this `extension`, **_all_** I **_had_** to **_do_** was
**_type_** `"rafce"` **_inside_** the `component file` and **_hit_** `return`.
For **_example_**, if I **_did this_** in a file **_called_** `Header.js`
**_inside_** my `components folder`, the **_following_** was **_generated_**:

```jsx
import React from 'react'

const Header = () => {
	return <div>Header</div>
}

export default Header
```
